# 🚀 Markdown 语法快速入门

这是一份为你量身定制的 Markdown 基础语法指南。Markdown 是一种轻量级的标记语言，能让你用纯文本轻松排出漂亮的排版。

---

## 1. 基础文本样式

在 Markdown 中，你可以通过简单的符号来改变文字的视觉权重：

*   **加粗**：在文字两边加上两个星号，例如 **这是加粗文字**。
*   *斜体*：在文字两边加上一个星号，例如 *这是斜体文字*。
*   ~~删除线~~：在文字两边加上两个波浪号，例如 ~~这是删除线~~。
*   '行内代码'：用反引号把文字包起来，适合放代码、快捷键或专有名词，例如 `` `Ctrl + C` ``。

---

## 2. 列表

无论是列出待办事项还是按顺序讲解，列表都非常实用：

### 无序列表（用星号 `*` 或减号 `-` 开头）
*   西红柿
    *   好吃
    *   好看
*   鸡蛋
*   小葱
*   鸡米花
*   哈哈哈

### 有序列表（数字加上一个点和一个空格）
1.  把锅烧热
2.  倒入鸡蛋液
3.  翻炒至熟
4.  放置盘中

## 4. 插入链接与图片

排版时怎么能少了超链接和插图呢？

*   **超链接**：语法是 `[文字](网址)`。
    *   例如：欢迎访问 [Markdown 官方教程](https://www.markdown.xyz)。
    *   例如: 欢迎访问 [百度网站](https://www.baidu.com)
*   **图片**：比链接只多了一个感叹号，语法是 `![图片描述](图片链接)`。
    *   例如：`![Markdown Logo](https://markdown-here.com/img/icon256.png)`
    *   例如: ![Markdown Logo](https://markdown-here.com/img/icon256.png)
